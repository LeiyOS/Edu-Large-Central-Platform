想象力大中台 — 静态预览包使用说明
================================

本包已打包为「单 HTML + 少量附属文件」，便于在浏览器中打开预览。

如何使用
--------
1. 推荐：将整个文件夹解压到本地，用 Chrome、Edge 或 Firefox 等现代浏览器双击打开 index.html（页面使用内联 ES Module，脚本与样式已打入该文件）。

2. 请保留与 index.html 同目录下的 favicon.svg、icons.svg 等文件（图标等资源）；勿只拷贝 index.html 单独挪动到其它路径，否则图标可能缺失。

3. 若安全软件拦截本地 HTML 或仍显示异常，可在本目录打开终端执行：
   python -m http.server 8080
   然后浏览器访问 http://localhost:8080

4. 正式部署到服务器：可将本目录内全部文件上传到静态站点；上线环境亦可只传 index.html（若不需要本地 favicon 可与运维约定使用线上图标）。

说明：index.html 体积较大属正常现象（含页面所需脚本与样式）。
