# Edu-Large-Central-Platform

想象力大中台项目

## 开发

基于 Vue 3、TypeScript 与 Vite。组件使用 `<script setup>`，详见 [script setup 文档](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup)。

更多工程与 IDE 说明见 [Vue 文档：TypeScript](https://vuejs.org/guide/typescript/overview.html#project-setup)。

```bash
npm install
npm run dev
```
